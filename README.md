# rwind

Tailwind-style utility classes for Roblox UI.

## Coverage

Status legend: ✅ shipped · ⚠️ shipped with caveats · 🕒 deferred (waiting on Roblox API) · 📦 planned · ❌ not expressible in Roblox

| Font weight       | `font-thin` → `font-black`                                                                                          | ✅      |                                                                                             | Full Tailwind scale                                                                                                     |

| Font size         | `text-xs` → `text-9xl`                                                                                              | ✅      |                                                                                             |                                                                                                                         |
| Line height       | `leading-none` → `leading-loose`, `leading-3` → `leading-10`                                                        | ✅      |                                                                                             |                                                                                                                         |
| Text color        | `text-{hue}-{shade}`, `text-white`, `text-black`, `text-transparent`                                                | ✅      |                                                                                             | Reads full palette                                                                                                      |
| Text alignment    | `text-left`, `text-center`, `text-right`, `align-top`, `align-middle`, `align-bottom`                               | ✅      |                                                                                             |                                                                                                                         |

| Opacity           | `opacity-*`                                                                                                         | 📦      | Decide v3 (`opacity-50`) vs v4 (`/50` modifier) syntax first                                | Roblox splits transparency between `BackgroundTransparency` and `TextTransparency` — class semantics need a design call |
| Effects           | `blur-*`, `brightness-*`, etc.                                                                                      | 📦      | Separate module; needs effect-instance strategy                                             | Roblox post-effects live on `Lighting`, not `GuiObject` — likely out of scope                                           |

## Palette

The full Tailwind v3 color palette ships out of the box: `slate`, `gray`, `zinc`, `neutral`, `stone`, `red`, `orange`, `amber`, `yellow`, `lime`, `green`, `emerald`, `teal`, `cyan`, `sky`, `blue`, `indigo`, `violet`, `purple`, `fuchsia`, `pink`, `rose` — each with shades `50` → `950`. Every color utility (`text-*`, `border-*`, `ring-*`, `shadow-*`) reads from the same palette.
